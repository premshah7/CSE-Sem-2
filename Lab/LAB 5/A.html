Concept of WWW

  • It stands for World Wie Web.
  • It is basically system of internet.
  • It is collection of text pages,videos, digitgal photographs,etc..
  • User access the World Wide Web facalities via client called a web browser.

Http Protocol

  1. It stands for Hyper Text Transfer Protocol.
  2. It is based on client-server architecture model and stateless request/resoponse protocol.
  3. 404 File not found is common HTTP status code.
